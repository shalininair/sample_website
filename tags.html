Important HTML tags

This page is designed as aquick reference guide for some of the common tags covered in Learn Enough HTML to be Dangerous. In the process of making it, we'll learn how to make HTML tables and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (level 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header include a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make bold text <strong>technical sophistication</strong> technical sophistication
a hyperlink make hyperlinks <a href = "...">technical sophistication</a> technical sophistication
img image embed an image <img src = "..." alt = "...">
code code defines a piece of computer code <code>array[].push("a")</code> array[].push("a")
